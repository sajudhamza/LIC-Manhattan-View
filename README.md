# LIC Manhattan View Hotel — Website

A boutique hotel website inspired by the Hyatt JDV style, for **LIC Manhattan View Hotel** in Long Island City, NY. Content is based on [licmanhattanviewhotel.com](https://www.licmanhattanviewhotel.com).

## Features

- **Hero** — Full-width hero with hotel name and “Book Direct” CTA
- **Rooms & Suites** — Carousel of room types (Manhattan Suite, Executive Suite, Premium Suite, Deluxe)
- **Amenities** — Wi-Fi, parking, fitness, restaurant, eco-friendly, accessibility
- **Eat & Drink** — Juquila Kitchen and Bar (Mexican cuisine)
- **Meetings & Gatherings** — Event inquiry and address (39-05 29th Street, LIC, NY 11101)
- **Gallery** — Image grid with lightbox
- **Footer** — Contact, quick links, booking link

## Tech Stack

- React 19 + Vite
- Tailwind CSS
- React Router
- Lucide React (icons)

## Setup

```bash
cd "LIC-Manhattan-View-Hotel"
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Build

```bash
npm run build
npm run preview
```
